Tilgjengelighetsfunksjoner i &brandFullName;

&brandShortName; har mange funksjoner som gjør nettleseren og nettet tilgjengelig for alle brukere, deriblant de som har dårlig syn, mangler syn eller har manglende evne til å bruke tastatur eller mus.

Bruk av tastatur

Tastatursnarveier

En liste over vanlige tastatursnarveier er tilgjengelig på &brandShortName; tastatursnarveier. På nettet kan du finne en sammenligning av tastatursnarveier slik de er implementert i &brandShortName;, Microsoft Internet Explorer og Opera.

Tillat tekst å bli merket med tastaturet

Tillat tekst å bli merket med tastaturet lar deg gå gjennom nettsideinnholdet som om det var i en tekstbehandler. Du kan bruke tastaturet for å merke nettsideinnhold og kopiere det til utklippstavlen. Du kan slå på denne funksjonen ved å velge &pref.menuPath;, åpne Generelt-fanen i Avansert-panelet, og deretter velge Tillat å velge tekst med tastaturet-innstillingen.

Du kan trykke på F7 for å slå denne funksjonen av eller på. Når du trykker F7 vil &brandShortName; spørre deg om du virkelig vil slå på denne funksjonen. Du kan slå av dette spørsmålet ved å velge Ikke vis denne dialogen igjen.

Søk etter tekst når jeg starter å skrive

Start søk når du starter å skrive lar deg raskt finne tekst eller hyperlenker i en nettside. Denne funksjonen har to moduser. Trykk / og start å skrive for å finne teksten blant all tekst på siden, eller trykk ' før teksten for å bare søke i hyperlenker. &brandShortName; vil utheve funnet tekst eller hyperlenke mens du skriver.

Følgende tastatursnarveier kontrollerer denne funksjonen:

  • Skriv inn flere tegn for å innsnevre søket. &brandShortName; vil utheve neste tekst eller hyperlenke, eller gi informasjon om at ingenting passet.
  • Trykk &accelKey;+G eller F3 for å finne neste forekomst av søketeksten.
  • Trykk &accelKey;+&shiftKey;+G eller &shiftKey;+F3 for å finne forrige forekomst av søkestrengen.
  • Trykk Backspace-tasten for å fjerne det siste tegnet du skrev inn. &brandShortName; vil gå til den forrige uthevede teksten.
  • Trykk Esc for å avbryte søket. Du kan også avbryte et søk ved å bytte fokus, bla på siden med musen eller tastaturet, eller vente til tidsavbruddet slik at søket forsvinner av seg selv.

Du kan endre oppførselen til denne funksjonen ved å velge &pref.menuPath;, åpne Generelt-fanen i Avansert-panelet, og velge Start søk når du begynner å skrive-innstillingen. Dersom denne innstillingen er valgt kan du når som helst starte et søk. Vanlig oppførsel er at automatisk startede søk vil lete etter all tekst på siden. Du kan fortsatt trykke ' eller / for å starte søk i valgt modus.

Du kan bruke denne funksjonen for å komme til knapper, tekstbokser og andre skjemakontroller. Søk for teksten umiddelbart før eller etter skjemakontrollen du vil nå, og trykk Tab. Når du trykker Tab vil fokusen settes til det neste skjemafeltet eller knappen etter den valgte teksten, og søket vil avsluttes.

Skrifttyper og farger

Overstyre skrifttyper på nettsider

Noen nettsider kan vise tekst med skrifttyper som er vanskelige å lese, mens andre nettsider benytter seg av nettleserens standard skrifttyper. Du kan sette nettleserens standard skrifttyper ved å gå til &pref.menuPath;, velge Innhold-panelet og velge en standard skrifttype og -størrelse i Skrifttyper og farger-seksjonen.

Du kan angi standard skrifttyper for ulike tekststiler, blant annet serif, sans-serif og monospace ved å trykke på Avansert-knappen. Mange nettsider angir sine egne skrifttyper, så valgene vil ha ingen effekt på slike nettsider. For å overstyre skrifttypene definert av slike sider kan du fjerne avkryssingen fra Tillat nettsider å bruke egne skrifttyper istedenfor de valgt over-innstillingen.

Vær oppmerksom på at noen nettsider kan vises feilaktig dersom de er avhengige av hva du har valgt som standard skrifttype.

Sette minimum skriftstørrelse

Noen nettsider kan vise skrift som er for liten til å lese enkelt. For å gjøre slike nettsider enklere å lese kan du sette en minimum skriftstørrelse. På nettsider som forsøker å vise skrift som er mindre enn denne størrelsen vil &brandShortName; gjøre teksten så stor som du angir.

For å sette minimum skriftstørrelse, gå til &pref.menuPath;, velg Innhold-panelet og trykk på Avansert-knappen i Skrifttyper og farger-seksjonen. Du kan angi en minimum skriftstørrelse fra nedtrekksmenyen.

Merk at noen nettsider vil muligens vises feil på skjermen, avehengig av verdien som er satt for minimum skriftstørrelse.

Endre størrelse på tekst i nettsider

Du kan midlertidig endre størrelsen på teksten på nettsider i Vis > Tekststørrelse-menyen. Den nye størrelsen vil være gjeldende inntil du lukker nettleservinduet eller fanen, selv om du besøker en annen nettside.

Følgende tastatursnarveier kontrollerer tekststørrelse:

  • Trykk &accelKey;++ for å øke tekststørrelsen. Denne forstørrer teksten til 120 % av dens opprinnelige størrelse, deretter 150%, 200%, 300%, 450%, 675%, 1013%, og 1519%.
  • Trykk &accelKey;+- for å gjøre tekststørrelsen mindre. Dette forminsker teksten til 90% av opprinnelig størrelse, deretter 50%, 33%, 22%, 15%, 10%, 7%, 4%, 3%, 2%, og 1%.
  • Trykk &accelKey;+0 for å gjenopprette tekststørrelsen til størrelsen som er nettsidens standard.

Merk at noen nettsider kan vises feilaktig dersom du øker eller forminsker tekststørrelsen.

Overstyre farger på nettsider

Noen nettsider kan vise tekst- og bakgrunnsfarger som er vanskelige å lese, og andre nettsider kan være avhengig av nettleseres standardfarger. Du kan sette standardfarger ved å gå til &pref.menuPath;, velge Innhold-panelet, og trykke på Farger-knappen i Skrifttyper & Farger-seksjonen. Du kan sette standard tekst- og bakgrunnsfarger i tillegg til standard farger for besøkte og ubesøkte hyperlenker. Du kan også angi om hyperlenker skal være understreket.

Mange nettsider har egne farger, så valgene du gjør vil ikke ha noen effekt for alle nettsider. For å overstyre fargene som er satt av mange nettsider må du fjerne avkryssingen fra innstillingen Tillat nettsider å velge egne farger, istedenfor valgene mine ovenfor.

Bruke et tema med høy kontrast

&brandShortName; oppdager automatisk om du bruker et tema med høy kontrast, og kan da vise alt innhold med høy kontrast. Dette overstyrer alle andre nettleser- eller nettsideinnstillinger, og påvirker &brandShortName;-grensesnittet også (alle menyer, vinduer og dialogvinduer) samt innholdet av nettsider du besøker.

For å bruke et tema med høy kontrast, gå til Startmenyen > Kontrollpanel > Tilgjengelighet > Skjerm og velg innstillingen Bruk høy kontrast.

Skru av Java applets

Noen nettsider gir en rik, interaktiv opplevelse ved å bruke Java™ applets. Dette kan gi problemer for noen brukere som er avhengig av å bruke tastaturet for å navigere. ettersom noen Java applets automatisk får tastaturfokus, og der fins ingen måte å byte ut av Java appleten og se resten av nettsiden. Dersom dette er et problem for deg, kan du slå av Java ved å gå til &pref.menuPath;, velge Innhold-panelet og fjerne avkryssingen på innstillingen Tillat Java.

Begrense JavaScript-oppførsel

Du kan slå av JavaScript ved å gå til &pref.menuPath;, velge Innhold-panelet og fjerne avkryssingen på Tillat JavaScript. Noen nettsider er avhengige av JavaScript og vil ikke virke ordentlig uten at JavaScript er tillatt.

Dersom du ikke ønsker å slå av JavaScript helt, fins det også flere avanserte JavaScript-innstillinger du kan bruke for å begrense oppførselen til skript. Trykk på Avansert-knappen rett etter Tillat JavaScript-valget for å åpne vinduet Avanserte JavaScript-innstillinger. Du kan bestemme om skript tillates å flytte eller endre vindusstørrelsen, minimere eller maksimere vinduer, slå av eller erstatte kontekstmenyer, skjule statuslinjen eller endre meldinger i statuslinjen.
